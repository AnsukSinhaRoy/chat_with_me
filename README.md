# Chat With Me

A browser-based AI twin chat app with two separate services:

- `frontend/`: Next.js browser app with responsive chat UI and voice-recording UX.
- `backend/`: FastAPI service for Gemini chat, Gemini transcription, and profile retrieval.

Read `VERCEL_DEPLOYMENT.md` before deployment. The frontend and backend should be deployed as separate Vercel projects with different root directories.
